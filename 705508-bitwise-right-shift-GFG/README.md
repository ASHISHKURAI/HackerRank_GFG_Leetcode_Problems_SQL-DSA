# [Bitwise Right Shift](https://www.geeksforgeeks.org/problems/bitwise-right-shift/1?page=3&category=CPP&sortBy=submissions)
## Easy
Given two positive integers a and b, perform a right bitwise shift of a by b positions and print the result.
Examples:
Input: a = 5, b = 2
Output: 1 
Explanation: 5 in binary is 101. Shifting right by 2 gives 001, which equals 1.
Input: a = 4, b = 1
Output: 2
Explanation: 4 in binary is 100. Shifting right by 1 gives 010, which equals 2.
