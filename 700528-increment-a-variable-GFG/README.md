# [Increment a variable](https://www.geeksforgeeks.org/problems/increment-a-variable/1?page=3&category=CPP&sortBy=submissions)
## Easy
Given a pointer to the variable a, complete the function updateVar(a)&nbsp;so that it increases the value of the variable by 10. The function should not return anything.Examples :  
Input: a = 4
Output: 14

Input: a = 61
Output: 71
Constraints:1&nbsp;≤ a&nbsp;≤&nbsp;103
